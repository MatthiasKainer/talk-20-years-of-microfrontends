Widgets

Widgets

Widgets - Pro/Cons

Easy to share, decent encapsulation of functionality

Can be deployed to a cdn independently

No clear style encapsulation, no maintainable state management

Didn't fit well into the dawn of monolithic frontend frameworks like angular and react

Has a great descendant called web components

SPA's and Framework components

Do support modularization

Web Applications lack the infrastructure to provide the same comfort for data/state management as microservices

It is easiest to maintain a single application with a shared, global state
